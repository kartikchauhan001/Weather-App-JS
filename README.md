# Weather-App-JS

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

Features :

1.Search weather by city name
2.Displays temperature in Celsius
3.Shows humidity percentage
4.Displays wind speed
5.Dynamic weather icons (Clear, Clouds, Rain, Drizzle, Mist)
6.Error handling for invalid city names
7.Modern UI with gradient card design
